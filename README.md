
# Chronix

Execution without chaos.

## Overview
Chronix is an autonomous productivity orchestration platform that helps users avoid missed deadlines using collaborative AI agents.

## Features
- Smart scheduling
- Rescue Mode
- Deadline prediction
- Dynamic prioritization
- Voice-assisted planning
- Focus mode
- Calendar sync

## Setup
```bash
pip install -r requirements.txt
```

Run backend:
```bash
uvicorn main:app --reload
```
